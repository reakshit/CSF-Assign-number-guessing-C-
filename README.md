# Number Guessing Game

★Project Description
This is a console-based number guessing game developed in C++. The computer generates a random number between 1 and 100, and the player must guess the correct number. The game provides feedback on whether the guess is "Too High" or "Too Low" and tracks the number of attempts made.

★Features
-> Random Number Generation: Ensures every game is unique by generating a new number each time.

->Live Feedback: Instantly tells the user if their guess is too high or too low.
